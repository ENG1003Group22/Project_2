Guide Run History
This application will help you records all your running activities, it will track your location and display your route on a map, with a green marker placed and your start point and a red marker placed at the end point. In the history tab, all saved runs will be shown, displaying the duration, distance, average speed and calories burnt of your run.

How to Record your Run :

1.Open the Run tab
2. Press 'Start' to run
3. Press 'Stop' when stop running
4. Press 'Clear' to clear the Map
5. Press 'Save' to save your route


How to display a Recorded Route :

1.Open the History Tab
2. Select any recorded run
2. Press 'Display' to display selected Route
3. Press 'Delete' to delete selected Route
