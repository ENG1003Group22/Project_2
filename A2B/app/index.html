JUST DO IT
Guide Run History
This application will help you records all your running activities, it will locate your place and show it on the map. It will give you a polylines from starting point to the end point of your run. In history tab it will show you how much calories you have burnt, Average speed of your run, running duration, total distance.

How to use guide :

1.Open Run tab
2. Press Start to run
3. Press Stop when stop running
4. Press Reset to start a new run
5. Press Save to save your running


History Guide :

1.Open History Tab
2. Select any recorded data
2. Press display to display selected data
3. Press delete to delete selected record
